# Online Voting System

## Description
The Online Voting System is a web-based application that enables users to participate in elections securely and efficiently. It allows registered users to log in, cast their vote, and view election results. The system also provides an admin panel to manage voters, candidates, and elections. The project is designed to ensure a simple, transparent, and user-friendly online voting process.

## Features
- User Registration and Login
- Secure Voting System
- One Vote Per User
- Candidate Management
- Admin Dashboard
- Election Result Display
- User-Friendly Interface

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## How to Run
1. Download or clone the repository.
2. Place the project folder in the XAMPP `htdocs` directory.
3. Import the database into MySQL using phpMyAdmin.
4. Start Apache and MySQL in XAMPP.
5. Open your browser and visit:
   ```
   http://localhost/your-project-folder
   ```
